# DataExportTool
Excel配置表导出工具  
支持插件自定义导出格式  
目前实现一种基于Excel到处自定义二进制的导出插件

第一行：注释  
第二行：KEY  
第三行：类型（bool byte short int string）  
第四行-最后：数据】
